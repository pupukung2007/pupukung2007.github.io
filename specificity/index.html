CSS Specificity

- Which Rule for the element

What is a CSS specification?

CSS specification is the way HTML choose which CSS rules in the .css stylesheets to apply to an element

How does it work?

Suppose you have multiple css stylesheet linked to a HTML page.
The HTML will load the stylesheets from top to bottom by the order of written.
Then, if any same CSS selector exists in multiple stylesheets,
The HTML will override the value of the properties with the latest loaded stylesheet and apply it.

A basic example

Button1 Button2

From above buttons, Button1 element uses .button1 CSS selector
and Button2 element uses .button2 CSS selector
These buttons are linked by 2 stylesheets, basic-style1.css and basic.style2.css, respectively.

/*basic-style1.css*/
.button1{
    border-style: solid;
    background-color: blue;
    color:red;
}
.button2{
    background-color: lime;
}

And...Next?

Button1 Button2
The HTML load basic-style2.css,
which include a yellow background for Button1, and a border for Button2.
Since Button1 already has a "blue" background from basic-style1.css,
It will be replaced with the "yellow" background from basic-style2.css.
And since Button2 has no border-style property defined in basic-style1.css,
The basic-style2.css will give Button2 its border.

So ... What Happened?

Button1 Button2
First, The HTML load basic-style1.css,
Giving Button1 a border, blue background, and red font.
It also give Button2 a lime(green) background.

/*basic-style2.css*/
.button1{
    border-style: solid;
    background-color: yellow;
}
.button2{
    border-style: solid;
}
